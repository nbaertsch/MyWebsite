JDOCS

Java Document Object Complexity Scanner

The Why

An overly complex DOM tree can have a number of negative implications for a webpage when combined with CSS styling that is too complex or his too wide a scope. See this article by Google for how a complex DOM tree can harm a pages performance.

The How

This application leverages the JSoup library to calculate the complexity of a single webpage. In this context, complexity specifically refers to:

  • Total Nodes
  • Max Node Depth
  • Number of Sibling Nodes
